README -- Analysis of AluJb overexpression functional assay data
The input data can be obtained from Figshare under DOI: 10.6084/m9.figshare.28928147
############################

  1. Analyze the Agilent Seahorse Mito Stress Test data:
    -Parse the Agilent Seahorse excel files to extract datapoints (Parse_Seahorse_Excel_IMR90_AluJb-OE_v3b.R, Parse_Seahorse_Excel_IMR90_AluJb-OE_functions_vJB01.R)
    -Statistically analyze and generate plots of the data (SUMMARY_Seahorse_Analysis_IMR90_AluJb_OE_Norm.R)

  2. Analyze the Propidium Iodide Cell Cycle data:
    -Statistically analyze and generate plots of the data (Analyze_Cell_Cycle_Results.R)

  3. Analyze the Proteostat Aggresome Detection Assay data:
    -Statistically analyze and generate plots of the data (Analyze_Proteostat_Results.R)
